# 请假管理系统

面向班主任的学生请假记录工具，收到家长请假通知后快速录入，随时查看谁在请假、统计请假情况，支持数据导出上报。

## 功能介绍

### 快速登记
收到家长微信/短信/电话通知后，5 秒内完成请假登记。支持事假、病假、公假三种类型，系统自动计算请假天数。

### 当前请假
一目了然看到今天有哪些学生正在请假，方便班主任掌握班级出勤情况。

### 记录查询
按学生姓名搜索，或按月份筛选，查看历史请假记录。

### 统计分析
查看班级请假数据：哪种请假类型最多、哪个学生请假最频繁、全年各月请假趋势。

### 数据导出
将指定年份的请假记录导出为 CSV 文件，方便上报给学校。

## 如何使用

### 首次启动

```bash
cd leave_tracker
pip install -r requirements.txt
python app.py
```

浏览器会自动打开，打开后即可使用。

### 修改班级名称

打开 `app.py`，找到第 40 行：

```python
CLASS_NAME = "初三(1)班"  # 改成你的班级
```

改成你自己的班级名称即可。

## 常见问题

**数据保存在哪里？**
所有记录保存在 `leave_records.db` 文件中，删除该文件会清空所有记录。

**需要联网吗？**
不需要，纯本地使用，不上传任何数据。

**如何备份数据？**
复制 `leave_records.db` 文件即可备份。
